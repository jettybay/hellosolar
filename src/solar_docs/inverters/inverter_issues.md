# Common Inverter Issues

## Inverter Not Turning On
- Battery disconnected
- Battery voltage too low
- Blown fuse

## Inverter Beeping Continuously
- Low battery
- Overload
- Overheating

## Inverter Shuts Down Suddenly
- Excessive load
- Faulty battery
- Internal protection activated

## Agent Action
Ask customer:
1. What appliances were on?
2. Battery age?
3. Error message on screen?
