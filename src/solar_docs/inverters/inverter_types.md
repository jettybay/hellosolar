# Types of Inverters

## 1. Off-Grid Inverters
- No connection to utility grid
- Uses batteries
- Ideal for areas with no power supply

## 2. Hybrid Inverters
- Works with solar, battery, and grid
- Automatic switching
- Most common in Nigeria

## 3. Grid-Tied Inverters
- Works only when grid is available
- No batteries
- Feeds excess power to grid
